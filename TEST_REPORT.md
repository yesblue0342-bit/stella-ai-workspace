6. **설정 반영**: B 무음 모드 → 팝업은 뜨되 소리 없음. 소리 모드 → 소리 포함.
- 실패 시 진단: 설정→알림→🩺 상태 확인(단계별 ✅/❌) + 📲 테스트(5초 후 셀프 푸시).
